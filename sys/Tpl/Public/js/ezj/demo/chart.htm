问答

  • 问：为何图表一直在 Loading？

    答：对于本演示，不能双击 HTML 文件打开演示，需放到 Web 服务器下，用 HTTP 协议来浏览。

  • 问：数据文件的 URL 是否可以含 QueryString 参数？

    答：可以，但 ?、& 等均需要 URLEncode。

  • 问：中文怎么会有乱码？

    答：请使用 UTF-8 编码。

  • 问：怎么不介绍数据文件？

    答：ezj.ext.chart 是调用的 Open Flash Chart，关于数据文件的更多介绍请参见 http://teethgrinder.co.uk/open-flash-chart-2/。
