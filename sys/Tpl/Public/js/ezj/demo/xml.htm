问答

  • 问：本演示有部分怎么在 Chrome 中无效？

    答：这是由于本演示有部分读取了菜单的 XML 文件，Chrome 中需要在 HTTP 协议下浏览，不能双击浏览。

  • 问：似乎 $x 的参数是不是既可以是 XML 文件地址，也可以是 XML 字符串，是这样吗？

    答：是的。

  • 问：进入 $x 的 XML 字符串是不是必须需要 XML 声明（<?xml）？

    答：不是的。可以要，也可以只是 XML 的一个片段，但必须只有一个根节点。
