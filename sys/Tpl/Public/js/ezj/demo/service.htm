问答

  • 问：ezj.ajax.service 是通过什么协议工作的？

    答：SOAP 1.1。

  • 问：ezj.ajax.service 可以访问第三方 Web Service 文件吗？

    答：ezj.ajax.service 运行于客户端，客户端浏览器基于安全的原因，不允许访问非本域的资源，所以想要在网站 A 通过 ezj.ajax.service 访问网站 B 的 Web Service 是不会有权限的，这跟 Ajax 一样。
